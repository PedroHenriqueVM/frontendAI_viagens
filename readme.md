

# ✈️ ViajaAI - Roteiros Inteligentes

![HTML5](https://img.shields.io/badge/HTML5-frontend-orange)
![TailwindCSS](https://img.shields.io/badge/TailwindCSS-3.x-blue)
![JavaScript](https://img.shields.io/badge/JavaScript-ES6-yellow)
![jsPDF](https://img.shields.io/badge/PDF-jsPDF-red)
![Status](https://img.shields.io/badge/status-online-brightgreen)

Aplicação web que gera roteiros de viagem personalizados usando inteligência artificial. Informe o destino, dias (até 15) e orçamento, e receba um roteiro completo com atrações, restaurantes, custos e dicas locais.

# 🌎 Demonstração

O projeto pode ser executado localmente abrindo o arquivo `index.html` em seu navegador.

---

# 📚 Funcionalidades

✔ Geração automática de roteiros personalizados
✔ Seleção de destino, dias (máx. 15) e orçamento
✔ Visualização detalhada do roteiro diário
✔ Dicas úteis para o destino
✔ Salvar roteiros favoritos no navegador
✔ Compartilhamento fácil (texto ou nativo)
✔ Download do roteiro em PDF

# ⚠️ Observações

- O roteiro gerado tem limite máximo de 15 dias.
- Este repositório contém apenas o frontend. A API de geração de roteiros não está inclusa.
- Para funcionamento completo, é necessário que a API esteja online.

---

# 🚀 Como usar

1. Abra o arquivo `index.html` em seu navegador.
2. Preencha o destino, número de dias (máx. 15) e orçamento.
3. Clique em "Gerar roteiro".
4. Visualize, salve, compartilhe ou baixe o roteiro em PDF.
